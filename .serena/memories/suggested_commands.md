# Suggested Commands

## Development
- `npm run dev` - Start development server
- `npm run build` / `next build` - Production build

## Quality Checks
- `npx tsc --noEmit` - TypeScript type checking (errors 0)
- `npx eslint .` - ESLint (errors 0)
- `next build` - Build verification

## System Utils (Darwin)
- `git` - Version control
- `ls`, `find`, `grep` - File system (prefer Serena tools)
